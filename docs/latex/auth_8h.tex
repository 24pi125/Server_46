Класс для аутентификации клиентов 

Содержит класс \mbox{\hyperlink{classAuthenticator}{Authenticator}}, который предоставляет статические методы для генерации соли, расчета MD5 хэшей и проверки подлинности клиентов.

\begin{DoxyNote}{Documentation}
Использует алгоритм MD5 для хэширования паролей с солью 
\end{DoxyNote}
\begin{DoxyWarning}{Documentation}
MD5 считается криптографически слабым, использовать только в тестовых целях 
\end{DoxyWarning}
