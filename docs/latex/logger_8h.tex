Класс для логирования 

Определяет класс \mbox{\hyperlink{classLogger}{Logger}} для ведения журнала событий сервера. Поддерживает запись логов в файл и вывод в консоль с различными уровнями важности (критические/некритические ошибки).

Особенности\+:
\begin{DoxyItemize}
\item Автоматическое добавление временных меток
\item Поддержка как полных строк, так и добавления к существующей строке
\item Разделение на обычные логи и ошибки 
\end{DoxyItemize}
