Класс для обработки клиентских сессий 

Определяет класс \mbox{\hyperlink{classSession}{Session}}, который обрабатывает отдельные клиентские подключения. Реализует полный цикл обработки\+: аутентификация, прием данных, вычисления, отправка результатов.

\begin{DoxyNote}{Documentation}
