Класс для обработки векторных вычислений 

Определяет класс \mbox{\hyperlink{classVectorProcessor}{Vector\+Processor}} для вычисления произведений элементов векторов с контролем переполнения. Предоставляет статические методы для работы как с отдельными векторами, так и с коллекциями векторов.
