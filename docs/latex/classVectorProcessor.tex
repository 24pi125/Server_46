Класс для векторных вычислений 

Специализируется на вычислении произведений элементов векторов с корректной обработкой переполнения 32-\/битных целых чисел.

\begin{DoxyNote}{Documentation}
Использует 64-\/битную арифметику для промежуточных вычислений

\end{DoxyNote}
