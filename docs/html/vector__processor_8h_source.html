23 * Специализируется на вычислении произведений элементов векторов
24 * с корректной обработкой переполнения 32-битных целых чисел.
25 *
26 * @note Использует 64-битную арифметику для промежуточных вычислений

29class VectorProcessor {
30public:
31 /**
32 * @brief Вычисляет произведение элементов вектора
33 *
34 * @param vector Входной вектор целых чисел
35 * @return int32_t Произведение элементов или граничное значение при переполнении
36 *
37 * @details
38 * Особенности:
39 * - Для пустого вектора возвращает 0
40 * - Использует 64-битные вычисления для предотвращения промежуточного переполнения
41 * - Проверяет переполнение перед каждым умножением
42 * - При переполнении возвращает INT32_MAX или INT32_MIN в зависимости от знака
43 *
44 * @note Время выполнения: O(n), где n - размер вектора
45 */
46 static int32_t calculate_product(const Vector& vector);

48 /**

52 * @return std::vector<int32_t> Вектор произведений для каждого входного вектора
53 *
54 * @details
55 * Применяет calculate_product() к каждому вектору в коллекции.
