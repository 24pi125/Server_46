Подробное описание

Класс обработки клиентской сессии

Обрабатывает полный цикл взаимодействия с клиентом:

  1. Прием и проверка аутентификационных данных
  2. Прием векторов для обработки
  3. Вычисление произведений элементов векторов
  4. Отправка результатов обратно клиенту
Предупреждения
Класс не является потокобезопасным, должен использоваться по одной сессии на поток

◆ process_vectors()

void Session::process_vectors ( )
private

Основная логика обработки векторов

Выполняет полный цикл обработки:

  1. Поиск и парсинг аутентификационных данных (48 hex символов)
  2. Проверка аутентификации
  3. Прием количества векторов
  4. Обработка каждого вектора
  5. Отправка результатов

Формат входных данных: [логин][16 hex соль][32 hex хэш][количество_векторов][вектор1]...[векторN]

Исключения
std::exceptionпри ошибках парсинга или сетевого взаимодействия

Проверяет аутентификацию клиента

Проверка аутентификации клиента

Аргументы
loginЛогин клиента
saltСоль для хэширования
received_hashПолученный хэш от клиента
Возвращает
bool true если аутентификация успешна
Аргументы
loginЛогин клиента
saltСоль (16 hex символов)
received_hashХэш от клиента (32 hex символа)
Возвращает
bool true если аутентификация успешна

Выполняет следующие проверки:

  1. Существование логина в базе
  2. Формат соли и хэша (длина и hex символы)
  3. Сравнение MD5(salt + stored_password) с полученным хэшем
Заметки
Детально логирует все шаги проверки для отладки
