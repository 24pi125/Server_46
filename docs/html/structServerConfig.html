Подробное описание

Структура конфигурации сервера

Содержит все настраиваемые параметры сервера:

  • путь к файлу базы данных клиентов
  • путь к файлу логов
  • порт для прослушивания подключений
Заметки
Все поля имеют значения по умолчанию
Примеры
/home/penza365/Server_46/src/main.cpp.

ServerConfig Заполненная структура конфигурации

Поддерживаемые аргументы: -h, –help Показать справку -p PORT Установить порт сервера -c, -d FILE Указать файл конфигурации клиентов -l FILE Указать файл логов

Исключения
std::invalid_argumentпри неверном формате аргументов
Аргументы
argcКоличество аргументов
argvМассив аргументов
Возвращает
ServerConfig Заполненная структура конфигурации

Обрабатывает следующие опции: -h, –help -> выводит справку и завершает программу -p PORT -> устанавливает порт сервера -c FILE -> указывает файл конфигурации клиентов -d FILE -> синоним для -c -l FILE -> указывает файл логов

Заметки
При неизвестном аргументе выводит справку и завершает программу с кодом 1
Если аргументов нет, возвращает конфигурацию по умолчанию
